# ข้อ 5 — Export รายงานเป็น Excel

## ทำอะไร
ปุ่มออกรายงานดาวน์โหลดเป็นไฟล์ Excel เปิดได้ทั้ง Excel และ Google Sheets

## ทดสอบยังไง
กดออกรายงาน -> เปิดไฟล์ใน Excel ได้ ตัวเลขตรงกับหน้าจอ

## 💡 เบื้องหลัง & เหตุผล

สร้างไฟล์ Excel **โดยไม่พึ่งไลบรารีภายนอกเลย** — เขียน `<table>` HTML แล้วส่งด้วย MIME `application/vnd.ms-excel` ซึ่ง Excel/Google Sheets เปิดได้เนทีฟ พร้อมหัวตารางจัดสีน้ำเงิน เลือกวิธีนี้เพราะเว็บเป็น static เปิดจากไฟล์ได้ทันทีไม่ต้องลงอะไร · _ข้อแลกที่รู้ตัว:_ ได้ .xls แบบ HTML-table ไม่ใช่ .xlsx แท้ที่ฝังสูตรได้ (ไว้ต่อยอดฝั่งเซิร์ฟเวอร์)
