# ข้อ 3 — ตัวกรองคิวโทร

## ทำอะไร
กรองคิวโทรด้วย checkbox 3 กลุ่ม (ผลการโทร / สถานะลูกค้า / แบรนด์) ทำงานร่วมกัน

## ทดสอบยังไง
เลือก ไม่รับสาย + รอติดตาม -> เห็นเฉพาะรายที่ตรงทุกเงื่อนไข

## 💡 เบื้องหลัง & เหตุผล

ตรรกะคือ **"AND ระหว่างกลุ่ม · OR ภายในกลุ่ม"** — เลือกหลายผลในกลุ่มเดียวเอามาทั้งหมด (OR) แต่ต้องผ่านทุกกลุ่มถึงโผล่ (AND) ตรงกับที่คนคิดจริง · "สถานะลูกค้า" **ไม่ได้เก็บเป็นฟิลด์** แต่ถอดสดจากข้อมูล (ยอดฝาก>0 = คืนชีพ · มีวันนัด = มีนัด · ติ๊ก dnc = ห้ามโทร) พนักงานจึงไม่ต้องกรอกสถานะเอง คิวสะท้อนความจริงให้อัตโนมัติ
