# ข้อ 12 — แจ้งเตือนผ่าน Telegram

## ทำอะไร
ตั้งค่า Token/Chat ID/เกณฑ์ + แจ้งทันทีเมื่อมีเหตุการณ์ + ส่งสรุปรายวัน/สัปดาห์ + mask เบอร์

## ทดสอบยังไง
ใส่ Token + Chat ID -> กดทดสอบส่ง -> ข้อความเข้ากลุ่ม Telegram

## 💡 เบื้องหลัง & เหตุผล

แยกปลายทาง 2 กลุ่ม (ทีม/หัวหน้า) + toggle เปิดปิดรายประเภท เพื่อกัน **"alert fatigue"** คนรับเฉพาะเรื่องที่เกี่ยวกับตัว · เบอร์ลูกค้าถูก **mask เป็น `081-xxx-1234`** ก่อนส่งเข้ากลุ่มแชต รักษาความเป็นส่วนตัว · _สำคัญสุด:_ ทุกการส่งห่อด้วย `try/catch` — Telegram ล่มหรือ Token ผิดจะแค่ warn ไม่ทำให้งานหลักพัง (แจ้งเตือนเป็น best-effort) · สรุปรายวัน/สัปดาห์ใช้ปุ่มกดแทน cron เพราะ static web ไม่มีตัวตั้งเวลาฝั่งเซิร์ฟเวอร์
