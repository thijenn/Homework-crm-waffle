# ข้อ 2 — Export รายชื่อลูกค้าเป็น CSV

## ทำอะไร
ส่งออกลูกค้าตามฟิลเตอร์เป็นไฟล์ CSV (UTF-8 BOM เปิด Excel ภาษาไทยไม่เพี้ยน เบอร์ขึ้นต้น 0 ไม่หาย)

## ทดสอบยังไง
กรองแบรนด์แล้ว Export -> ในไฟล์มีเฉพาะแบรนด์นั้น · เปิดใน Excel ภาษาไทยอ่านได้

## 💡 เบื้องหลัง & เหตุผล

แก้ 2 จุดที่ CSV + Excel ภาษาไทยมักพังเสมอ: (1) ใส่ **BOM `EF BB BF`** นำหน้าไฟล์ Excel จึงตีความเป็น UTF-8 ภาษาไทยไม่กลายเป็นต่างดาว · (2) เบอร์โทรเก็บเป็น **"ข้อความ" ตลอด** ไม่เคยถูกแปลงเป็นตัวเลข เลข 0 นำหน้าจึงไม่หาย · ใส่ quote แบบ RFC 4180 เฉพาะช่องที่มี `, " \n` และคอลัมน์ทั้ง 17 ตรงกับฟอร์แมตนำเข้า → **export แล้ว import กลับได้ครบ** ใช้เป็นไฟล์สำรองได้ในตัว
