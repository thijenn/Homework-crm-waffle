CRM Dashboard — เว็บไซต์ติดตามลูกค้าขาดฝาก
================================================

วิธีเปิดใช้ (เลือกอย่างใดอย่างหนึ่ง)
------------------------------------
1) ง่ายสุด: ดับเบิลคลิกไฟล์  index.html  เปิดในเบราว์เซอร์ได้ทันที
   (ไม่ต้องต่อเน็ต ไม่ต้องลงโปรแกรม)

2) ถ้าต้องการเปิดผ่าน server (เช่นจะแชร์ในวง LAN):
   - เปิด Terminal มาที่โฟลเดอร์ site นี้
   - พิมพ์:  node server.js
   - เปิดเบราว์เซอร์ไปที่  http://localhost:8731


ไฟล์ในโฟลเดอร์
----------------
index.html  = หน้าเว็บ
styles.css  = สไตล์/ดีไซน์
app.js      = ตรรกะแดชบอร์ด (กราฟ ตาราง ฟิลเตอร์)
data.js     = ข้อมูลที่ดึงจากไฟล์ Excel เดือนมิถุนายน
server.js   = เซิร์ฟเวอร์ทางเลือก (ไม่บังคับ)


อัปเดตข้อมูลเดือนใหม่
----------------------
data.js ถูกสร้างจากไฟล์ Excel ต้นทาง ถ้ามีข้อมูลเดือนใหม่
แจ้งให้ช่วยรันสคริปต์ดึงข้อมูลใหม่ แล้วแทนที่ data.js ไฟล์เดียว
หน้าเว็บที่เหลือใช้ซ้ำได้เลย


เข้าสู่ระบบ (Login)
--------------------
เมื่อเปิดเว็บจะเจอหน้าเข้าสู่ระบบก่อน
  ชื่อผู้ใช้ (Username) : admin
  รหัสผ่าน  (Password) : admin123

* รหัสผ่านเก็บไว้ในเบราว์เซอร์ของเครื่องนี้ (localStorage)
  เมื่อเปลี่ยนรหัสแล้วจะจำค่าใหม่ไว้ ครั้งต่อไปต้องใช้รหัสใหม่
* ออกจากระบบได้ที่ปุ่ม ⏻ มุมล่างซ้าย


9 หน้าในเว็บ
-------------
- ภาพรวม         : KPI + โดนัทผลการโทร + Top5 VIP + ไฮไลต์ + กราฟยอดฝาก/ROI + funnel
- ลูกค้าควรโทรก่อน: จัดอันดับลูกค้าที่ควรโทร (สูง/กลาง/ต่ำ) จากสัญญาณสนใจจริง
- เทียบรายแบรนด์  : ตารางเทียบทุกแบรนด์ (คลิกหัวคอลัมน์เพื่อเรียง)
- แนวโน้มรายวัน   : กราฟยอดฝากกลับรายวัน 1-31 + วันที่ทำยอดดีสุด
- เทียบรายเดือน   : สะสม snapshot แต่ละเดือน + เทียบ delta (นำเข้าหลายเดือนเพื่อเทียบ)
- ผลการโทร       : โดนัท + แท่งสรุปผลการโทร (รวมคำพิมพ์ผิดให้แล้ว)
- รายชื่อลูกค้า   : ค้นหา/ฟิลเตอร์ + แก้ไข/เพิ่มลูกค้า + ฟิลด์ใหม่ (พนักงาน/สาเหตุ/นัดถัดไป)
- นำเข้าข้อมูล    : อัปโหลด CSV อัปเดตทั้งระบบ + ส่งออก/เทมเพลต + คืนค่าต้นฉบับ
- ตั้งค่า         : เปลี่ยนรหัสผ่าน + จัดการผู้ใช้หลายคน (admin/พนักงาน)

ปุ่มบนสุดขวา "🖨️ พิมพ์ / PDF" = สั่งพิมพ์หรือบันทึกหน้าเป็น PDF


Shortcut บน Desktop
--------------------
- "เปิด CRM Dashboard.command" = ดับเบิลคลิกเปิดเว็บ
- "สำรอง CRM.command"          = ดับเบิลคลิกสำรองข้อมูลเป็น ZIP ลงวันที่อัตโนมัติ


ระบบหลายผู้ใช้ + สิทธิ์
----------------------
- admin : เห็นทุกอย่าง + จัดการผู้ใช้ (เพิ่ม/ลบ) ได้
- พนักงาน (agent) : ใช้งานแดชบอร์ดได้ แต่จัดการผู้ใช้ไม่ได้
- เพิ่มผู้ใช้ใหม่ได้ที่ ตั้งค่า > จัดการผู้ใช้


การแก้ไข/นำเข้าข้อมูล จะถูกจดจำไว้ (localStorage)
- กดปุ่ม "ส่งออกข้อมูลปัจจุบัน" ในหน้านำเข้า เพื่อเซฟ CSV กันหายเสมอ

ผู้ใช้ทดสอบ (3 สิทธิ์):
  admin / admin1234   → ADMIN (เห็นทุกอย่าง + Audit + จัดการผู้ใช้)
  head1 / head1234    → SUPERVISOR (เพิ่มรายงาน/วิเคราะห์/Telegram)
  agent1 / agent1234  → AGENT (เห็นเฉพาะคิว/ลูกค้า/SMS ของตัวเอง)

ฟีเจอร์ครบ 12 ข้อ: เปลี่ยนรหัส, Export CSV, ตัวกรองคิว, รายงานสัปดาห์/เดือน,
Export Excel, นัดโทรกลับ, ห้ามโทร(DNC)+ประวัติ, ผลงานพนักงาน, Cohort+เทียบโปร,
Audit Log, คลัง SMS, แจ้งเตือน Telegram
